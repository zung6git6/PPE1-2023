Présentation

  • Le mini-projet consiste à nous faire familiariser avec l'extraction des informations des URLS à partir d'un fichier de texte d'URLS.
  • En pratique, nous avons des fichiers où se trouvent un URL par ligne et nous allons manipuler le fichier dans l'environnement de bash.
  • Beaucoup de techniques de bash sont concernées. La première à prendre en compte est les conditions d'exécution. Comme nous prenons le fichier URLS comme argument 1, nous devons poser des conditions afin d'assurer que l'utilisateur entre bien un fichier et pas n'importe quoi.
  • Ensuite, la deuxième technique est d'employer une boucle while au fichier dans le but de numéroter les liens, d'avoir les codes HTTP ainsi que les types d'encodage
  • Quant aux codes HTTP, on va faire ainsi : $(curl -s -I -L -w "%{http_code}" -o /dev/null $line)
  • Concernant les types d'encodage, $(curl -s -I -L -w "%{content_type}" -o /dev/null $line | grep -E -o "charset=\S+" | cut -d"=" -f2 | tail -1)
  • Finalement, nous allons transformer les résultats en tableau HTML
